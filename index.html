Projetos

Alurageek

Descrição

Projeto de um E-commerce de produtos Geek/Nerd. Onde é possivel visualizar, pesquisar por nome e ver por categoria; além de possuir sistema login para administradores, que podem adicionar, remover e editar produtos. O projeto foi feito com o React, Typescript e vanilla CSS. Foram utilizados bibliotecas como: react-router-dom, para as rotas; mirajejs, para simular um backend; e CSS modules para ajudar na estilização dos componentes.
